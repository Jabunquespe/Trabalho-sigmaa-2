Este aplicativo nasceu com o objetivo de simplificar a vida acadêmica dos estudantes do IFRS. Sabendo como a navegação no portal original do SIGAA pode ser lenta ou complexa quando acessada pelo celular, desenvolvemos esta interface alternativa focada em desempenho e usabilidade.

Utilizando o estilo visual Neo-Brutalista, trouxemos um design com forte personalidade, alto contraste e carregamento instantâneo das informações que mais importam no dia a dia: suas notas e suas faltas.
⚙️ Tecnologias Utilizadas

O ecossistema do projeto foi dividido de forma leve e eficiente:

    🟢 Node.js & Express Responsáveis por erguer o nosso servidor local e gerenciar as rotas seguras da API de integração.

    🟠 Tough-Cookie & Axios Criam um "navegador virtual" no back-end para gerenciar sessões e cookies do SIGAA automaticamente, burlando quedas de conexão.

    🔴 Cheerio (Web Scraping) Uma biblioteca ultra veloz que varre o código do portal governamental, raspa os dados brutos e os organiza para exibição.

    🔵 Neo-Brutalist UI (HTML5 & CSS3) Interface responsiva pura, livre de frameworks pesados, garantindo transições fluidas e visual moderno de cards interativos expansíveis.
